# Laboratorium 1
## Razor Pages

Pierwsze podejście do technologii Razor Pages - strona w pełni tłumaczona na język polski posiadająca działający CRUD dla 4 modeli.

### Stack
+ Web API
    - .Net 5.0
    - Razor Pages
    - Entity Framework Core Code-First
    - Baza Danych Sql (wykonanie w oparciu o SQL Server 2019 Express)

### Startup
+ ISS Express - przed uruchomieniem aplikacji należy wykonać w terminalu polecenie update-database które utworzy potrzebną bazę danych - pod warunkiem podania właściwego connection stringa "DefaultConnection" w appsettings.json kierującego do lokalnej instancji serwera SQL
